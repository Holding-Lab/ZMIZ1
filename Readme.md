## Quick Start

Below is a quick example that will generate the figures.

```R
# Install Package
library(devtools)
install_github("andrewholding/ZMIZ1")
# You may also need to install packages from BioConductor e.g. Vulcan

# Load package
library(ZMIZ1)

figure1b()
figure2a()
figure2b()
figure3a()
figureS1()
```
